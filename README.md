# Commandix POC

Fundação técnica do projeto baseada em Monólito Modular.

## Requisitos

- Node.js (v20+)
- pnpm (v8+)
- Docker e Docker Compose

## Instalação

```bash
# Instala as dependências de todo o workspace
pnpm install
```

## Executando com Docker Compose

Sobe banco de dados, API e Frontend:

```bash
docker compose up --build
```
